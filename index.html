GIT COMMANDS FOR BRANCHING

To create Branch 01, key in the command in bash terminal

To create new branch : git branch [branch name]

To assess new branch : git checkout [branch name]
Make sure you have done a git add . and commit before you switch back to master

To commit new branch : git push -u origin [branch name]

To go back to main branch : git checkout master

Merging Branch into Master

To merge the branch into master, you can do a git merge..
Remember to do the merge in the master branch or the destination branch (usually is Master)

To merge [g-branch-01] : git merge [branch name]
